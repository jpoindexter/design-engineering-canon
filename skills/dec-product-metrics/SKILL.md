# Metrics & Outcomes

Frameworks to define and measure value — so you optimize outcomes, not output.

- **North Star Framework:** One metric capturing the core value delivered, with input metrics that move it. Aligns the team on a single number.
- **Google HEART:** Happiness, Engagement, Adoption, Retention, Task success — paired with **Goals → Signals → Metrics** to turn each dimension into something instrumentable.
- **AARRR (Pirate Metrics):** Acquisition → Activation → Retention → Referral → Revenue. The funnel for finding where users drop.
- **OKRs:** Objectives (qualitative direction) + Key Results (measurable outcomes). Outcomes, not output.

## How to apply

- Pick one **North Star** that genuinely reflects user value (not vanity reach); wire input metrics beneath it.
- Use **HEART → Goals/Signals/Metrics** to avoid measuring what's easy instead of what matters.
- Walk the **AARRR** funnel to locate the weakest stage before optimizing anything else.
- Write **Key Results** as outcomes ("activation +15%"), never as shipped features ("launch X").
